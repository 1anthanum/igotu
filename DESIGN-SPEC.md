# IGOTU 设计方案 v2

## 核心理念

**"像植物一样，慢慢生长"**
整个应用以 **生物光（Bioluminescence）** 为视觉基底——深色背景中透出柔和的自然光，
配色随情绪分数动态变化，AI 对话可以实时影响全局氛围。

---

## 1. 情绪驱动配色系统

情绪分数 1-5 对应五种自然光色，颜色自动渗透到全局所有 UI 元素：

| 分数 | 情绪 | 主色调 | 光晕 | 隐喻 |
|------|------|--------|------|------|
| 1 | 很低落 😢 | `#8b5cf6` 紫罗兰 | 冷紫 | 深夜的萤火虫 |
| 2 | 不太好 😕 | `#6366f1` 靛蓝 | 蓝紫 | 黎明前的微光 |
| 3 | 一般 😐 | `#14b8a6` 青绿 | 青色 | 安静的苔藓 |
| 4 | 还不错 🙂 | `#10b981` 翠绿 | 绿色 | 新叶舒展 |
| 5 | 很好 😊 | `#f59e0b` 暖琥珀 | 金色 | 阳光穿过树冠 |

> **设计原则**：低分时颜色更冷更暗（包裹感，不刺激），高分时颜色更暖更亮（开放感，庆祝但不过度）。

---

## 2. 颜色更新的两个触发源

### 2a. 手动情绪记录（已实现）
用户在情绪记录页选择分数 → `moodStore.log()` → `moodTheme.setMood(score)` → CSS 变量更新

### 2b. AI 对话实时分析（新增）
AI 在对话过程中分析用户情绪状态 → 后端返回 `mood_score` 字段 → 前端接收后调用 `moodTheme.setMood(score)` → UI 随对话进程缓缓变化

**实现方式**：
- 后端：Claude API 响应中附加 `estimated_mood: number` 字段
- 前端：ChatView 收到消息后检查该字段，若存在则平滑过渡到新颜色
- 过渡动画：CSS transition 2-3 秒，让颜色变化像日出/日落一样自然
- **不打断对话**：颜色变化完全在背景发生，用户不会被弹窗或提示分散注意力

## 3. 视觉基底：C · 生物光

### 3.1 背景与空间
- **主背景**：`#060f0d`（极深墨绿近黑）
- **卡片背景**：`rgba(10,25,20,0.6)` + `border: 1px solid rgba(mood-accent, 0.06)`
- 卡片顶部有一条 2px 的光线（`linear-gradient` 使用 mood-accent 色）
- **留白充足**：每个元素之间有呼吸空间，不拥挤

### 3.2 卡片与组件
- 圆角 16px，无强烈阴影
- Hover 时边框微微亮起（mood-accent 色 15% 透明度）
- 光晕效果用 `box-shadow: 0 0 Xpx mood-glow`，强度随情绪变

### 3.3 动效原则
- **生长感**：元素出现时 `translateY(20px) → 0`，像植物破土
- **呼吸感**：关键元素（呼吸练习圆圈、AI 头像）有 `scale(0.95-1.05)` 的缓慢脉动
- **无突变**：所有颜色变化走 `transition: 2-3s ease`，避免闪烁
- **低能量友好**：动画轻柔缓慢，不使用快速弹跳或闪烁

### 3.4 文案语气
- 温柔但不讨好，像一个安静的朋友
- 使用植物/自然隐喻：种子、花园、生长、光、根
- **不用**："加油"、"你一定可以"、"坚持就是胜利"这类正能量口号
- **用**："慢慢来"、"这样就好"、"你在这里就够了"

## 4. 各页面设计要点

### 4.1 登录页
- 极深墨绿背景 + 漂浮的生物光粒子（5-6 个小光点，缓慢呼吸明灭）
- 叶脉图标 + "IGOTU" 字标
- 标语："像植物一样，慢慢生长"
- 输入框：半透明深绿底 + 情绪色描边
- 按钮文案："走进花园"（登录）/ "种下第一颗种子"（注册）

### 4.2 首页仪表盘
- 问候区域："你今天还好吗？" + 时间段问候
- 一条生物光横线（mood-accent 渐变）分隔问候与内容
- **情绪选择器**：5 个圆形种子，hover 时发出对应颜色的光晕
- 卡片内容：
  - "🌱 今日已种下 X 颗种子"（微成就计数）
  - "🍃 呼吸空间"（快捷进入呼吸练习）
  - "💭 和你的伙伴聊聊"（进入 AI 对话）
- 底部：简洁的周趋势缩略图

### 4.3 情绪记录页
- **光谱选择器**：5 个柱状条，高度不同，颜色从紫→绿→金渐变
- 选中后柱条发光 + 展开一个笔记输入区域
- 历史视图：每天一个圆点，颜色对应当天情绪，形成一条"情绪脉搏线"
- 洞察卡片：以自然隐喻描述模式（如"周三的散步像阳光一样带来了变化"）

### 4.4 AI 对话页
- AI 名称："小树"（🌱）
- AI 头像：圆形 + 情绪色边框 + 呼吸脉动动画
- 在线状态：绿色呼吸小圆点 + "在线陪伴"
- Bot 气泡：深绿底 + 情绪色微描边
- User 气泡：情绪色 30% 透明底
- **内嵌式练习**：对话中可直接嵌入呼吸练习圆圈（不跳转页面）
- 快捷建议：底部 3 个 tag 式按钮，文案温和（"先做个深呼吸"、"写下此刻感受"）
- **颜色联动**：AI 每次回复如果附带 mood_score，背景色缓缓过渡

### 4.5 工具箱（呼吸/接地/认知重构/PHQ-9）
- 保持已有的全屏沉浸式呼吸练习
- 将光环颜色从当前的固定配色改为 mood-accent 驱动
- 所有练习都有"随时离开"的温柔退出按钮

### 4.6 分析/数据页
- 图表用 mood-accent 色系
- 趋势图背景与网格线用极低透明度白色
- 统计数字用 mood-accent 高亮

### 4.7 设置页
- 品牌保持 "IGOTU"
- 所有文字用 CSS 变量控制

## 6. 不做什么（设计红线）

- ❌ 连续打卡提示 / 断签惩罚 / 排行榜
- ❌ "你已经 X 天没使用了" 的内疚式推送
- ❌ 鲜艳高饱和的颜色
- ❌ 快速弹跳/闪烁动画
- ❌ "加油！你一定可以！" 等空洞正能量
- ❌ 强制完成流程才能退出
- ❌ 在抑郁低分时展示"对比数据"暗示退步
